Workflow #1: Landing Page Design

Category: Design & Branding | Time: 2-4 hours | Output: Production-ready landing page

Transform a project idea into a polished, professional landing page using AI-assisted research, design generation, and asset creation.

Step 1: Competitive Research

Research Phase
Find the best existing examples

Claude

  • Ask Claude to research businesses similar to your project
  • Get a list of 10-15 comparable products/services
  • Review each one - note what works in terms of:
    • Visual design and aesthetics
    • Information hierarchy
    • Call-to-action placement
    • Feature presentation
  • Pick the top 2-3 that feel right

Step 2: Design Direction

Visual Exploration
Generate initial design concepts

Claude Midjourney

  • Give your top picks back to Claude
  • Ask Claude to generate Midjourney prompts based on these references
  • Run 10-20 prompts in Midjourney
  • Look for the visual style that resonates
  • Select 2-3 directions you like
Why Midjourney?
Midjourney is currently the best tool for getting an early design feel. It captures mood, style, and aesthetic direction faster than any other approach.

Step 3: Brand Kit Development

Iteration Phase
Develop comprehensive brand kit

Claude (subagent)

  • Give your selected design direction back to Claude
  • Ask for 5-10 brand kit variations
  • Important: Use a subagent or separate instance to avoid filling your main context window
  • Iterate in rounds:
    • Round 1: General design direction (colors, typography, mood)
    • Round 2: Individual features (buttons, cards, headers)
    • Round 3: Specific components as needed
  • Spin up additional instances for parallel exploration if needed

Step 4: Asset Creation

Production Phase
Create and prepare visual assets

Claude Midjourney Photoshop

  • Claude generates individual HTML mockups for each section
  • For custom assets (backdrops, illustrations, hero images):
    • Ask Claude for Midjourney prompts
    • Generate in Midjourney
    • Download the ones you like
  • Use Photoshop to:
    • Crop out backgrounds (fastest method)
    • Clean up edges
    • Export in correct formats
  • Save everything to an /assets folder that Claude can access

Step 5: Final Assembly

Build Phase
Assemble the final landing page

Claude

  • Finalize the brand kit based on all iterations
  • Claude builds the complete landing page using:
    • The finalized brand kit
    • All assets from /assets folder
    • Best practices from your research
  • Result: Production-ready landing page
Context Window Management
The key to this workflow is keeping your main Claude instance clean. Use subagents for:
  • Brand kit iterations (5-10 variations each round)
  • Individual component design
  • Asset prompt generation
Your main instance stays focused on decisions and final assembly.

Folder Structure

Recommended project setup
project-name/
├── research/
│   └── competitors.md       # Notes on researched businesses
├── design/
│   ├── midjourney/          # Saved Midjourney outputs
│   ├── mockups/             # HTML section mockups
│   └── brand-kit.md         # Final brand decisions
├── assets/
│   ├── images/              # Processed images
│   ├── icons/               # Icon assets
│   └── backgrounds/         # Background images
└── output/
    └── index.html           # Final landing page
